# Live Weather Dashboard 🌤️

Real-time weather data and 5-day forecast
for any city in the world.

## Live Demo
[Click here](https://weather-dashboard-webapp-p8m6fdarafezqu65trefqp.streamlit.app)

## Features
- Current weather with all key metrics
- 5-day temperature and humidity forecast
- Daily summary cards with weather icons
- Sunrise and sunset times
- Compare weather across multiple cities
- Includes Indian cities quick select

## Tools Used
- Python, Streamlit, OpenWeatherMap API
- Requests, Pandas, Matplotlib

## How to Run Locally
pip install streamlit requests pandas numpy matplotlib
Add API key to .streamlit/secrets.toml
streamlit run app.py
